The Computer
Motherboard The motherboard is the center of the computer, It allows all the parts to connect and communicate with each other to work together. It also has a bios which allows you to configure hardware. Such as adjusting the cpu core voltages, cpu base clock speeds and cpu multipliers.

CPU [Central Processing Unit] This is basically the component that calculates constantly for your computer to run. It it executes instructions given by operating system or program or etc. cpus can range from having single processing cores to as many as you want. The common standard of today is a quad core. There are also many other properties to a cpu such as cache amount, pin count for example.

RAM [Random Access Memory] This is pretty much a small amount of extremely fast storage to use for random data when a program needs it to be stored to be called upon later. Usually now these days people don't use ram as general storage unless they want the contents to delete when the computer ram is disconnected from power. Back in the day people used to create ram drives to speed up their systems but those methods are now obsolete.

GPU [Graphics Processing Unit/Graphics Accelerator/Graphics Card] This is a processor that is optimised for rendering graphical items such as your browser and screen with a graphical user interface to video games to rendering CGI movies. This component was invented because beforehand the cpu was rendering images and it was really slow. So the graphical accelerator was born to render things really fast so the cpu has less load and graphics will be produced at an optimized rate.

HDD [Hard Disk Drive] It is a mechanical storage that physically moves to read and write data. This is where you want your permanent data to be stored. This is also a slower storage compared to ram but it is permanent non volatile storage unlike ram.

Peripherals [Mouse/Keyboard/Drawing Tablet] Aka HIDs(Human interface devices). HIDs are devices that require interaction to interface to your computer. On a interaction such as a keypress or mouse movement the device sends a signal to the computer. The computer needs a driver to read the signal and then tell the operating system that that interaction has occurred so it can do whatever it is programmed to do for that interaction.
