The Computer
Motherboard The motherboard is the center of the computer, It allows all the parts to connect and communicate with eachother to work together.

CPU [Centeral Processing Unit] This is basically the component that calculates constantly for your computer to run. It based off the intrustions of a program it tells what information to save in the Ram, What it needs to load from the hard drive, What graphics the gpu needs to render and ect.

RAM [Random Access Memory] This is pretty much a small amount of extremely fast storage to use for random data when a program needs it to be stored to be called opon later.

GPU [Graphics Processing Unit/Graphics Accelerator/Graphics Card] This is a processor that is optimised for rendering graphics such as your browser and screen with a ui to video games to rendering CGI movies.

HDD [Hard Disk Drive] This is where the os and progams are stored. This is a slower storage compared to ram but it is permanant storage unlike ram.

Peripherals [Mouse/Keyboard/Drawing Tablet] These are methods of inputs to interface to your computer. On a interaction such as a keypress or mouse movement the device sends a singal to the computer. The compter needs a driver to read the singal and to register the interaction on the program or os.
